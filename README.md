### PianoJs

Just written in Basic Javascript to have fun and refresh my CSS and DOM Handling.

## Features

- Cloned Piano by CSS
- Piano keys play according notes:
  - by mouse click
  - by keyboard event
- Autoplay songs:
  - jingle bells
  - happy birthday
- Speed adjustable
